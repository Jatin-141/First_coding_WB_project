# Amazon Homepage Clone (HTML & CSS)

This project is a static clone of the Amazon homepage built using **HTML and CSS** as part of self-learning and hands-on practice.

It represents my **first project in my coding journey**, created to understand webpage structure, layout design, and styling by recreating a real-world website interface.

## Features
- Amazon-style navigation bar
- Search bar with icon integration
- Product grid layout
- Footer with multiple sections
- Responsive layout using Flexbox

## Tech Stack
- HTML5
- CSS3
- Font Awesome icons
